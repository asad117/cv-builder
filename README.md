# CV.AI | Minimalist Resume Builder

**CV.AI** is a high-performance, minimalist web application designed to help professionals turn raw experience into polished, ATS-friendly resumes in seconds. Built for the modern job seeker, it focuses on clean design and intelligent formatting rather than complex manual editing.

## ✨ Key Features

*   **Magic Format:** Paste your rough notes or an old resume, and let the AI instantly structure it into a professional layout.
*   **Minimalist Templates:** A curated selection of high-impact, recruiter-approved templates that prioritize readability.
*   **Live Preview:** Real-time side-by-side view of your changes.
*   **Multi-Format Export:** Download your resume as a professional PDF or a Microsoft Word (.doc) file.
*   **Privacy First:** Your data is processed for the build and stays under your control.

## 🚀 Tech Stack

*   **Framework:** Next.js 14/15 (App Router)
*   **Styling:** Tailwind CSS
*   **State Management:** Zustand
*   **PDF Generation:** React-to-Print / Custom Print CSS
*   **Icons:** Lucide React
